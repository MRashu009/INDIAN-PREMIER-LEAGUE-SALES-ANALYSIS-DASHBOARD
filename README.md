# Indian Premier League (IPL) Analysis Dashboard

This project is an interactive Excel dashboard created to analyze IPL match data from 2008 to 2018.  
The dashboard provides insights into team performance, match results, venues, and player achievements.

## Dashboard Overview

The dashboard includes the following key information:

- IPL Season Analysis
- Winning Team of the Season
- Runner-Up Team
- Player of the Series

Example (2018 Season):
- Winner: Chennai Super Kings
- Runner-Up: Sunrisers Hyderabad
- Player of the Series: Sunil Narine

## Key Insights

The dashboard analyzes multiple aspects of IPL data such as:

- Matches won by teams when batting first vs fielding first
- Toss decision based winning percentage
- Top venues with highest number of matches
- Teams with most IPL titles
- Top 10 Man of the Match award winners

## Visualizations Used

The dashboard uses different charts to present insights:

- Stacked Column Chart (Matches won by team)
- Donut Chart (Toss decision winning percentage)
- Bar Chart (Top 10 Man of the Match winners)
- Stacked Bar Chart (Top 10 venues)
- Treemap (IPL Title Winners)

## Interactive Features

The dashboard includes filters for:

- IPL Season (2008 – 2018)

This allows users to analyze data season-wise.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
